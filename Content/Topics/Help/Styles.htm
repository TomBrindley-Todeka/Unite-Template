Styles

Character and Paragraph Styles are contained in a Cascading Style Sheet called: Unite Master Style Sheet.CSS

This is file is read only. In future, it is likely that this style sheet will be held on a central server and downloaded automatically whenever a document is issued. This means you cannot change or create styles.

If you need a new style, contact your Flare administrator.

Endnotes

To add external references, for example to web searches, use an endnote My new footnote.

Position the cursor where you want the reference number to appear.

Select Insert/Footnote from the ribbon.

Select Endnotes Proxy.

Click OK.

The footnote appears at the bottom of the page (it will not be present on printed output).

Overtype "My new footnote" with the cross reference text.

Highlight the text an select Insert/Hyperlink.

Type the URL of the page to be referenced.

The end note will appear in a table at the end of the document (achieved using an EndNote proxy, contained in the Endnote topic.)
