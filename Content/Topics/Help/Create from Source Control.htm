Create from Source Control

The master template is held in a Github repository (in the cloud).

To create a new Threat Paper, a new project is cloned from the master.

  1. Select File>New Project>Import from Source Control

    2. The Import wizard is started.

    The path to the Remote repository should be pre-populated, along with your Name and email. If not, type the Remote, Name and Email. Leave Save Per: as User.

  2. Click Next.

  3. Select the Browse button.

  4. Click on to expand the file list.

  5. Click on the .flprj file and select OK.

    6. The repository is cloned. The project name remains as the original, so it is best to rename it. This must be done outside of Flare.

  6. Close the current project (File>Close>Close Project).

  7. Open Windows Explorer and browse to the MyProjects folder (usually Documents/My Projects/)

  8. Find the newly created local repository folder and open it.

  9. Rename the .flprj file.

  10. Return to Flare and choose File>Open.

  11. Browse to the new project and double click on the .flprj file.

On opening you will be invited to commit new files, choose yes.
